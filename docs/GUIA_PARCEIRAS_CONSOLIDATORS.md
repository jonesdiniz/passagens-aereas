# 🤝 Guia de Parcerias com Consolidators e Acesso a Tarifas Secretas

## O que são Tarifas Secretas?

Tarifas secretas (ou privadas) são preços de passagens aéreas que **não estão disponíveis ao público geral**. Elas são negociadas entre:

- Companhias aéreas e consolidators (atacadistas de passagens)
- Companhias aéreas e grandes corporações (tarifas corporativas)
- Companhias aéreas e agências de viagens com alto volume

Estas tarifas podem oferecer descontos de **10% a 60%** em relação ao preço público.

## 🏢 Consolidators no Brasil — Contatos e Estratégia

### 1. Central de Emissão de Passagens (Grupo Fly)

- **Site:** <https://centraldeemissao.com.br/>
- **Descrição:** Fornece passagens e milhas para 1.500+ agências mensalmente
- **Serviços:** Emissão de passagens, acesso a milhas, tecnologia
- **Como contatar:**
  1. Acesse o site e clique em "Fale Conosco"
  2. Mencione que você tem uma plataforma digital de captação de clientes
  3. Pergunte sobre programa de parceria para agências/afiliados
  4. Negocie comissão por venda convertida
- **Dica:** Eles têm plano individual (R$58/mês) — faça primeiro para entender o modelo

### 2. Fly Miles (Grupo Fly)

- **Site:** <https://flymiles.com.br/> (verificar)
- **Descrição:** Consolidator de milhas e passagens
- **Serviços:** Venda de milhas, emissão de passagens com milhas
- **Estratégia:** Oferecer fluxo de leads qualificados em troca de comissão

### 3. Moblix

- **Site:** <https://www.moblix.com.br/>
- **Descrição:** Plataforma brasileira com API de voos
- **Serviços:** API de busca e emissão de passagens
- **Vantagem:** Tem API REST! Pode ser integrada tecnicamente
- **Como contatar:**
  1. Preencha formulário de contato no site
  2. Pergunte sobre API para agências/parceiros
  3. Solicite documentação técnica e pricing

### 4. TravelSoho (Flight Aggregator)

- **Site:** <https://www.travelsoho.com/>
- **Descrição:** Integrador de GDS, APIs e conteúdo NDC
- **Serviços:** Agregação de múltiplas fontes de voos
- **Estratégia:** Pode intermediar acesso a múltiplos consolidators

### 5. Iddas Agência

- **Site:** <https://agencia.iddas.com.br/>
- **Descrição:** Plataforma para agências de viagens
- **Serviços:** Integração com fornecedores, emissão
- **Documentação:** <https://agencia.iddas.com.br/documentacao>

## 📧 Script de Abordagem para Consolidators

Use este modelo de email ao contatar consolidators:

```
Assunto: Proposta de Parceria — Plataforma de Captação de Leads para Passagens Aéreas

Olá [Nome],

Meu nome é [Seu Nome] e sou fundador do GPS de Passagens Aéreas,
uma plataforma digital que ajuda usuários a encontrar as melhores
tarifas utilizando estratégias avançadas de busca.

Temos captado diariamente leads qualificados de pessoas buscando
passagens internacionais e nacionais, muitas interessadas em tarifas
privadas e emissão por consolidator.

Gostaria de explorar uma parceria onde encaminhamos esses leads
para vocês e recebemos comissão por cotação convertida em venda.

Nosso sistema já gera briefing automático com:
- Rota, datas e número de passageiros
- Tarifa pública de referência (benchmark)
- Dados de contato do cliente
- Prioridade da busca

Podemos agendar uma call rápida para discutir?

Abraços,
[Seu Nome]
[Seu WhatsApp]
[Seu Email]
```

## 🔄 Fluxo de Trabalho Recomendado

```
1. USUÁRIO busca no seu site
   ↓
2. SISTEMA mostra tarifa pública + estratégias
   ↓
3. USUÁRIO clica "Solicitar Tarifa Secreta"
   ↓
4. SISTEMA captura dados (nome, email, rota, datas, benchmark)
   ↓
5. ADMIN (você) recebe notificação no painel
   ↓
6. VOCÊ envia briefing para consolidator (copiar do painel)
   ↓
7. CONSOLIDATOR responde com tarifa secreta
   ↓
8. VOCÊ cadastra tarifa no painel admin
   ↓
9. SISTEMA mostra tarifa secreta ao usuário (notificação futura)
   ↓
10. USUÁRIO aceita → você fecha venda com consolidator
```

## 💰 Modelos de Comissão Negociáveis

| Modelo | Descrição | Quando Usar |
|--------|-----------|-------------|
| **Por lead** | R$ 5-20 por lead enviado | Início da parceria, baixo volume |
| **Por cotação** | R$ 30-100 por cotação respondida | Lead qualificado, interesse confirmado |
| **Por venda** | 5-15% do valor da passagem | Parceria consolidada, alta confiança |
| **Mensalidade + %** | R$ 200-500/mês + 3-8% | Volume alto, exclusividade |

## ⚠️ Considerações Legais

1. **CNPJ:** Para emissão em nome próprio, considere abrir CNPJ de agência de viagens
2. **IATA:** Não é obrigatório para começar, mas facilita com grandes consolidators
3. **Reclame Aqui:** Monitore a reputação dos consolidators antes de firmar parceria
4. **Contrato:** Sempre formalize a parceria por escrito, mesmo que simples
